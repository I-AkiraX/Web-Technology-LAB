Try

The "try" keyword is used to specify a block where we should place exception code. The code which is prone to exceptions is placed in the try block. When an exception occurs, that exception occurred is handled by catch block associated with it. The try block must be followed by either catch or finally. It means, we can't use try block alone.

Sample Code:

public class JavaExceptionExample{
 public static void main(String args[]){
 try{
   int data=100/0;
 }catch(ArithmeticException e){
  System.out.println(e);
 }
 System.out.println("rest of the code...");
 }
}

Output:

 Exception in thread "main" java.lang.ArithmeticException: / by zero
